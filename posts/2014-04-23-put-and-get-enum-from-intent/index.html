Intent に Enum を詰めて、取り出す

タイトルの通りです。

public enum MyTypes {
	One,
	Two,
	Three
}
intent.putExtra("hoge",MyType.Two);
MyTypes t = (MyTypes)intent.getSerializableExtra("hoge");

取り出すのは getSerializableExtra であることに注意。

なんか int に変換されそうじゃん、と思って getIntExtra を使うと取り出せない。
